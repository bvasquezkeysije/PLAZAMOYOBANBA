1. ANÁLISIS DE VULNERABILIDADES
    

Consiste en identificar fallas, brechas o debilidades técnicas del sistema, por ejemplo, considere lo siguiente: Escaneo con herramientas (Nessus, OpenVAS, Nmap), Registro de vulnerabilidades detectadas, Clasificación según severidad, Propuesta de corrección. Puede utilizar tablas. CONSIDERE EN ANEXOS EVIDENCIAS DE TODOS LOS HALLAZGOS.

  

Ejemplo de tabla:

|                                 |           |                   |                    |           |
| ------------------------------- | --------- | ----------------- | ------------------ | --------- |
| Vulnerabilidad                  | Severidad | Herramienta usada | Evidencia          | Estado    |
| Puertos abiertos sin protección | Alta      | Nmap              | Captura de escaneo | Cerrado   |
| Credenciales débiles            | Alta      | Hydra             | Informe de prueba  | Corregido |
